# Team Builder

A web app for instructors to create teams of students based on constraints.

## Features

- Add students by name
- Add tags to students (e.g., math, science)
- Add relations: specify which students or groups should or shouldn't be together, with importance levels
- Generate teams that best satisfy the constraints

## Usage

Open `index.html` in a browser.

## Tests

Run `npm test` in the `test` directory.
